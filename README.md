# API Sample

API of a Rent a Car to explain backend concepts on the Topic 8 of the course.

Based on Django & Django REST Framework

DB: PostgreSQL

## Requirements

* pip
* virtualenv 

## Run development server

```console
> .\scripts\dev.bat
```
## Prepare migrations

```console
> .\scripts\makemigrations.bat
```

## Apply migrations

```console
> .\scripts\migrate.bat
```
